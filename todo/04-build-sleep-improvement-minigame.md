# Build mini-game for improving light sleep scores

Game starts at the user's actual sleep score. Core mechanic:

## Rhythmic tapping/swiping for resonant breathing (0.1 Hz / ~6 breaths/min)

- Game levels (especially N1 "stormy" zones) require tapping in sync with visual/auditory cue
- Expanding circle at 5–6 sec inhale/exhale (like 4-7-8 pattern)
- This is HRV biofeedback — users practice resonant breathing to boost vagal tone and reduce sympathetic arousal

## Evidence
- Mobile HRV-BF with 0.1 Hz breathing improves subjective sleep quality, vagal activity, and cardiorespiratory function during sleep
- Short sessions before bed lower pre-sleep arousal and may reduce fragmentation
